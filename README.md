# Hamilton Traffic Speed Hump Finder
<!--StAuth10244: I Jacqueline Robinson, 000967585 certify that this material is my original work. No other person's work has been used without due acknowledgement. I have not made my work available to anyone else.-->

A web mapping app for visualizing traffic speed humps across Hamilton

## Running locally

In the project directory, run python3 -m http.server 8000 (if on a mac), then open the index.html page in browser

If this does not work, there's a live deployment on https://jazzy-paprenjak-87d576.netlify.app/ for the same files
